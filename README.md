# dev-blog

개인 기술 블로그 (Next.js + React + TypeScript).

## 실행

의존성 설치 후 실행:

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속.

## 현재 포함된 것

- `app/layout.tsx`: 헤더/네비/푸터가 있는 공통 레이아웃
- `app/page.tsx`: 홈(히어로 섹션)
- `app/posts/page.tsx`: 글 목록(플레이스홀더)
- `app/about/page.tsx`: 소개(플레이스홀더)
- `app/globals.css`: 전역 스타일
